# Style Pillars

## Core Style
**Stylized social logistics city**

This game should look like a place players want to work in, return to, and show to friends. It should read clearly in motion, feel lived-in, and support a repeatable night-shift logistics fantasy.

## Pillars
1. **Readable First**
- silhouettes matter
- roads, bays, boards, and prompts should be obvious
- avoid muddy clutter and visually noisy realism

2. **Cozy Industrial**
- practical infrastructure
- warm depot lighting against cooler night streets
- inviting social hub, harsher route spaces

3. **Wet Urban Night**
- asphalt, concrete, metal, warning lights, neon accents
- moody but not dark enough to hurt readability

4. **Status In The World**
- public boards
- branded trucks
- visible ranking/progression displays
- hub spaces that show aspiration and activity

5. **District Personality**
- each district must read instantly from palette, landmarking, signage, and lane structure

6. **Roblox-Native Cleanliness**
- strong simple forms
- stylized proportions
- clean materials
- legible color groupings

## Anti-Goals
- generic empty industrial graybox forever
- muddy realism
- random creator-store clutter
- overdecorated roads that hide gameplay affordances
